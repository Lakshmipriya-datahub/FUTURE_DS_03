Marketing Funnel & Conversion Performance Analysis

📌Project Overview

Marketing Funnel & Conversion Performance Analysis focuses on analyzing E-commerce User Behaviour Dataset to extract meaningful insights that support strategic decision-making. The primary objective of this E-Commerce User Behavior Analysis project is to analyze customer interactions across the sales funnel in order to identify drop-off points, evaluate channel effectiveness, and improve overall conversion performance. This dashboard provides a comprehensive analysis of customer behavior across the e-commerce funnel, highlighting user drop-offs, channel performance and revenue trends.

🎯Business Objectives

A E-Commerce Platform wants to analyze customer purchasing patterns and business performance to:

Analyze marketing funnel data to identify conversion drop offs

Analyze channel performance and opportunities to improve lead-to-customer conversion

Support data-driven decision-making

Build an interactive dashboard for business insights

📂 Dataset Information

The dataset contains key business metrics such as:

event_time

Project Workflow

1.Data Loading in Excel

The dataset was imported into Excel for initial cleaning and validation.

Key activities:

Inspecting dataset structure

Understanding data types

Identifying missing values

Performing preliminary data checks

2.Data Cleaning & Preparation

Removed duplicate records

Handled missing values

Converted date columns into proper format

Standardized categorical values

3.Power BI Dashboard

An interactive Power BI dashboard was developed to present insights in a business-friendly format.

Dashboard Features:

KPI Cards (Total Revenue, Total Profit, Total Quantity)

Channel-wise Customers 

User Funnel

Time-based filtering using slicers

📈 Key Insights

• Major drop-off at Add-to-Cart stage (~85%)

• Referral channel brings highest revenue and best-quality leads

• Paid Ads perform the weakest among channels

• Electronics is the top revenue generating category

• Early funnel stages need optimization for better conversions
